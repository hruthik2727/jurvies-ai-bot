## 🎯 How to Use

### Start the Server:
```bash
cd jurvies_ai_bot
python main.py
```

### Open a UI:
Double-click any HTML file:
- `grok_client.html`
- `chatgpt_style.html`
- `advanced_client.html`
- `test_client.html`

### Start Chatting:
1. Click a suggestion chip
2. Or type your own message
3. Press Enter to send
4. Get helpful, friendly responses!
